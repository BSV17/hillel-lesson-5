Типы селекторов

  • Селекторы по тэгам: h1. Будут выбраны все элементы с таким тегом
  • Селекторы по id: #type. Будет выбран только элемент с таким id
  • Селекторы по классам: .class . Будут выбраны все элементы с таким классом
  • Селектор по нескольким классам: .class1.class2 . Будут выбраны элементы которые имеют оба класса
  • Универсальный селектор: *. Будут выбраны все элементы веб-страницы
  • Вложенные селекторы: .class1 .class2 . Будут выбраны элементы вложенные в другой элемент
  • Дочерние селекторы: .class1 > .class2 . Будут выбраны элементы первого уровня вложенности
  • Соседние селекторы: .class1 + .class2 . Будет выбран элемент с .class2, который стоит непосредственно после элемента c .class1, если у них общий родитель
  • Родственные селекторы: .class1 ~ .class2 . Будут выбраны все соседние элементы для элемента с .class1, которые имеют .class2
  • Селекторы атрибутов: input[type="text"] . Будут выбраны элементы с определенными атрибутами
